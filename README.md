# ExamScheduler

A simple Java console program that demonstrates basic OOP concepts (Inheritance, Polymorphism, and Interfaces)
by allowing the user to add and view exam schedules (Online and Offline).

## Project Structure

```
ExamScheduler_GitHub/
├─ ExamScheduler.java
├─ README.md
└─ .gitignore
```

## Files
- `ExamScheduler.java` - Single-file Java program containing all classes (Exam, OnlineExam, OfflineExam, and main class ExamScheduler).
- `README.md` - This file.
- `.gitignore` - Recommended ignores for compiled files.

## How to compile and run

1. Open terminal / command prompt and navigate to the project directory.
2. Compile:
   ```
   javac ExamScheduler.java
   ```
   This produces `ExamScheduler.class` and other class files.
3. Run:
   ```
   java ExamScheduler
   ```

## Usage
- Choose option `1` to add an exam (enter subject, date, time, and type).
- For Online exams, provide the meeting link.
- For Offline exams, provide the room number.
- Choose option `2` to view all scheduled exams.
- Choose option `3` to exit.
